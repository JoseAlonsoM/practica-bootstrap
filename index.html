Grid de Bootstrap (12 col/pos)

Sirven para estructurar, a nivel de diseño, la información. Deben estar dentro de un container. Existen containers de dos tipos:

1 Contenedor ancho fijo "container bg-primary text-light"

2 Contenedor ancho fluído "container-fluid bg-primary text-light"


Usando col sin interfijo

Inmediatamente dentro del container debemos incluir filas, es decir, div’s con el class row. Cada fila tiene como máximo (a no ser que lo modifiquemos), 12 posiciones.

1 "col" dentro del mismo "row"
2 "col" dentro del mismo "row"
3 "col" dentro del mismo "row"

1div class="container bg-primary text-light" --- > 2div class="row" --- > 3div class="col" (x3) Una fila con tres columnas, por tanto, cada columna distribuidas en el grid ocupará 4 posiciones.

Breakpoints de la rejilla de Bootstrap 5

Small (teléfonos en formato landscape) .col-sm ≥576px

Extra extra large (ordenadores de sobremesa con pantalla más grande) .col-xxl ≥1400px

Iconos

Bootstrap 5 no viene con un pack de iconos instalados por defecto.
Sin embargo, sí que existe una colección de iconos, específicos de Bootstrap 5 que pueden ser añadidos de manera independiente.
Para ello, tenemos varias opciones.
1 Añadir la siguiente CDN a la ya existente de Bootstrap:
link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.5.0/font/bootstrap-icons.css"
2 Instalar las fuentes usando node:
npm i bootstrap-icons
link rel="stylesheet" href="./node_modules//bootstrap-icons/font/bootstrap-icons.css"
